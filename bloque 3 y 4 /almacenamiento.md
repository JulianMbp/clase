


# Almacenamiento Fundamental — Resumen (Módulo 4)

## Introducción

En esta clase se introduce el **concepto de almacenamiento en entornos cloud** y se explica cómo funcionan las **capas gratuitas de las plataformas de nube**.

El instructor utiliza una **analogía con la educación pública y privada**, donde ambas ofrecen acceso al conocimiento aunque con diferentes costos o características.

En el caso del **cloud computing**, la mayoría de plataformas ofrecen **una capa gratuita**, pero es importante entender que **no todos los servicios son gratuitos ni lo son para siempre**. 

---

# Conceptos Clave del Entorno Cloud

## 1. Capas gratuitas en la nube

Todas las plataformas de cloud computing ofrecen un **nivel gratuito (Free Tier)** que permite:

* Aprender y practicar con servicios reales.
* Crear recursos para laboratorios o ejercicios.
* Acceder a ciertos servicios sin costo durante un tiempo limitado o de forma permanente.

Sin embargo:

* No todos los servicios están incluidos.
* Algunos servicios solo son gratuitos por **12 meses**.
* Otros tienen un **límite de uso mensual**.

---

# Registro en Plataformas Cloud

Para utilizar las plataformas cloud normalmente se requiere:

* Crear una cuenta.
* Registrar una **tarjeta de crédito**.

### Recomendación del instructor

Utilizar **tarjetas recargables** o de bajo saldo para evitar cargos inesperados.

Motivo:

* La nube puede hacer **un cargo temporal de verificación**.
* Ese cargo suele ser **reembolsado después de algunas horas**.
* Solo se cobraría dinero si se utilizan servicios **fuera de la capa gratuita**.

La academia **no asumirá costos generados por el uso de la nube**, por lo que cada estudiante debe usar los servicios con responsabilidad. 

---

# Buenas Prácticas al Usar la Nube

Para evitar costos:

* Usar únicamente **servicios incluidos en la capa gratuita**.
* **Eliminar recursos creados** después de cada práctica.
* Supervisar constantemente los servicios activos.

Ejemplo:

Las **máquinas virtuales** pueden ofrecer hasta **700 horas mensuales gratuitas**, lo que permite mantenerlas encendidas todo el mes dentro del límite permitido.

# Principales Proveedores de Cloud

Durante la clase se presentan varias plataformas de nube.

## AWS (Amazon Web Services)

Es una de las plataformas cloud más grandes del mundo.

Servicios gratuitos disponibles incluyen:

* **EC2** → máquinas virtuales (cómputo)
* **S3** → almacenamiento de objetos
* **RDS** → bases de datos relacionales
* **DynamoDB** → bases de datos NoSQL
* **Servicios serverless**

La capa gratuita permite practicar y realizar laboratorios.

---

## Microsoft Azure

Ofrece:

* **12 meses de servicios gratuitos**
* **Más de 65 servicios gratuitos permanentes**

Entre los servicios disponibles:

* Máquinas virtuales Linux y Windows
* Inteligencia artificial
* Bases de datos
* Contenedores
* Redes

Los estudiantes con **correo institucional** pueden recibir créditos adicionales.

---

## Google Cloud Platform (GCP)

GCP también ofrece:

* Créditos iniciales para nuevos usuarios
* Un nivel gratuito con varios servicios

Servicios destacados:

* **Cloud Storage** (almacenamiento)
* **Compute Engine** (máquinas virtuales)
* **Kubernetes y contenedores**
* Herramientas de inteligencia artificial

---

## IBM Cloud

IBM es fuerte en:

* Bases de datos
* Servicios empresariales
* Escalabilidad para sistemas bancarios

Ofrece:

* Más de **40 servicios gratuitos**
* Servicios de **cómputo, almacenamiento, bases de datos e inteligencia artificial (Watson)**.

---

## Oracle Cloud (OCI)

Oracle también ofrece servicios cloud enfocados en:

* almacenamiento
* bases de datos
* infraestructura empresarial

Además ha realizado alianzas para ejecutar modelos de **inteligencia artificial**.

---

# Equivalencia de Servicios entre Nubes

Muchos servicios de diferentes nubes **cumplen funciones similares**, aunque tengan nombres distintos.

Ejemplos:

| Función                  | AWS | Azure              | GCP            |
| ------------------------ | --- | ------------------ | -------------- |
| Almacenamiento           | S3  | Azure Blob Storage | Cloud Storage  |
| Base de datos relacional | RDS | Azure SQL          | AlloyDB        |
| Cómputo                  | EC2 | Virtual Machines   | Compute Engine |

Esto significa que **aprender una nube facilita aprender las demás**.

# Conclusiones

* Todas las plataformas cloud ofrecen **capas gratuitas para aprendizaje**.
* Es fundamental **usar los recursos de manera responsable**.
* Registrar una tarjeta de crédito es necesario, pero se recomienda usar **tarjetas recargables**.
* Las principales nubes del mercado son **AWS, Azure, GCP, IBM y Oracle**.
* Muchos servicios son **equivalentes entre plataformas**, por lo que aprender una facilita entender las demás.
* La práctica directa en las plataformas es esencial para aprender cloud computing.
